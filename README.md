# Sistemas Distribuídos

## 📌 Descrição
Repositório contendo uma aplicação distribuída desenvolvida na disciplina de Sistemas Distribuídos da Universidade Federal de Ouro Preto (UFOP).

O projeto utiliza containers Docker para simular um ambiente distribuído, com múltiplos serviços interagindo entre si.

---

## 🧠 Conceitos abordados

- Computação distribuída
- Comunicação entre serviços
- Tolerância a falhas
- Replicação
- Containerização
- Orquestração com Docker Compose

---

## 🛠️ Tecnologias utilizadas

- Node.js
- Docker
- Docker Compose

---

## ⚙️ Arquitetura

A aplicação é composta por múltiplos containers que se comunicam através de rede interna, simulando um ambiente distribuído.
